# **App Name**: CollabHub

## Core Features:

- Project Posting: Companies can create detailed project postings including title, description, tech stack, budget, and deadline.
- Developer Application: Developers can browse open projects and apply with a proposal text, managed through the Supabase PostgreSQL database.
- Shortlisting: Companies can shortlist applicants based on their proposals, triggering notifications and enabling design submissions.
- Design Submission: Shortlisted developers can submit designs either via Figma link or file upload through Supabase Storage. The application allows comparing different design submissions, before a company chooses a winner.
- Compensation Tracking: Track compensation for the winning developer and participation compensation for the non-selected shortlisted developers.
- Real-time Project Chat: Enable real-time communication between companies and developers within each project using Supabase subscriptions.
- AI-Powered Proposal Tool: AI tool assists developers to formulate compelling project proposals using insights from successful applications and project requirements.

## Style Guidelines:

- Primary color: Slate Blue (#708090) to convey professionalism and trustworthiness.
- Background color: Light Gray (#D3D3D3) to offer a clean, neutral backdrop.
- Accent color: Dark Orange (#FF8C00) for important CTAs and highlights to draw user attention.
- Headline font: 'Space Grotesk' sans-serif for a tech-forward feel; body text: 'Inter' sans-serif for readability. 
- Code font: 'Source Code Pro' for displaying code snippets.
- Use modern, line-based icons to represent various project categories and actions.
- Implement a responsive grid layout with clear content sections for both company and developer dashboards.
- Incorporate subtle transition animations and loading states to enhance the user experience.
